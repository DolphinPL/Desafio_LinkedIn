- **Sobre o Projeto**
   - Elaborar um gerenciador de tarefas com as seguintes funcionalidades:

     - Ver tarefa(s) adicionada
     - Atualizar tarefas
     - Completar tarefas
     - Deletar tarefas que foram completadas
     - Sair
